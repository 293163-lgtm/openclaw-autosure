# Release v0.1.0

Autosure is a self-contained OpenClaw skill for safe unattended continuation.

This first public release includes:

- bounded and unlimited `/autosure` continuation controls
- failure-aware auto-resume with circuit governance
- optional UI capsule userscript for direct controls
- native fallback patch path for local Control UI environments where userscripts do not execute reliably
- deploy, repair, doctor, and runtime validation scripts

## Notes

- Default product route remains the userscript UI capsule.
- On mini49, native fallback is the locally preferred working path because Tampermonkey on local Control UI has been unreliable.
- `/autosure N` now starts the first frontstage continuation immediately, then continues remaining rounds after each completed turn.
