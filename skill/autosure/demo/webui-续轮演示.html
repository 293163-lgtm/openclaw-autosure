Autosure 续轮 · Web UI 可视化演示

这是独立模拟页，用来展示「答完一轮 → 自动再发一条用户消息（轮换 发动词.txt）→ 模型继续」的视觉效果；不是 OpenClaw 内置聊天页本身。

真实环境：你在 OpenClaw Web 里发 /autosure N，网关加载插件后，由插件触发与这里「自动用户气泡」等价的流程。 更推荐：在 OpenClaw 真实对话里发整行 /autosure-demo，由插件在会话内做可视化（见 autosure-resume 插件）。
打开方式：在访达中双击本文件，或用浏览器打开该 HTML 路径。若要改演示速度，可编辑本文件中的 STEP_MS 常量。
